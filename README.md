## 🚨 Forking this repo
If you wanna fork this repo, feel free to do so.

## 🛠️ Installation & Set Up
1. Install dependencies
```bash
npm install
```
2. Start the development server
```bash
npm start
```

## 🚀 Deploy
To deploy a static site:
1. Run the build script
```bash
npm run build
```
